Web は複雑になりすぎました。

ページを開くたびに、何十もの外部スクリプトが走り、トラッキングが行われ、フレームワークが初期化されます。ブログを書くだけなのに、なぜこれほど重いのか。

avel はその問いへの答えのひとつです。

書いたことが伝われば十分です。そのために JavaScript は必要ありません。おしゃれなアニメーションも、リアルタイム更新も、いりません。テキストと、最低限の構造と、読めるフォントがあれば、ブログは成立します。

Zola は Rust 製の静的サイトジェネレーターです。ビルドが速く、依存関係がなく、生成される HTML はシンプルです。avel はその Zola の上に、さらに何も足さない方向で作られています。

引き算の美学、とでも言えばいいでしょうか。
