avel は [阿部寛のホームページ](http://www.abesan.jp/) にインスパイアされた、JS ゼロの軽量 [Zola](https://www.getzola.org/) テーマです。

## 特徴

- **JavaScript ゼロ** — ページ表示に JS を一切使いません
- **外部リソースなし** — デフォルト設定では CDN も Google Fonts も読み込みません
- **インライン CSS** — スタイルシートファイルすら不要。HTML に直接埋め込まれます
- **`config.toml` だけで完結** — HTML や CSS を触らずに全スタイルを調整できます

## なぜ JS ゼロなのか

高速・プライバシー・シンプルさ。この3つが理由です。

JS がなければトラッキングスクリプトも混入しません。外部リソースがなければ CDN 障害の影響も受けません。シンプルなほど壊れにくく、長く使えます。

阿部寛さんのホームページが何年も変わらず存在し続けているのは、シンプルだからです。
