[阿部寛のホームページ](http://www.abesan.jp/) にインスパイアされた JS ゼロの [Zola](https://www.getzola.org/) テーマ。

- JavaScript ゼロ
- 外部リソースなし（デフォルト設定）
- CSS はインライン
- `config.toml` だけで全スタイルを制御

## なぜ JS ゼロか

速い。プライベート。壊れない。

JS がなければトラッキングスクリプトは入らない。外部リソースがなければ CDN 障害の影響を受けない。シンプルなものは長持ちする。

阿部寛のホームページが何年も変わらず存在し続けているのはシンプルだからだ。
