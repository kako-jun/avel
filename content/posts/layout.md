Several values control the page layout.

```toml
[extra]
max_width = "960px"
content_align = "left"
line_height = "1.5"
title_align = "center"
main_padding = "1em"
```

`max_width` keeps long lines readable on wide screens. If you do not set these values, avel uses its default look.
